O que é HTML?

O HTML(HyperText Markup Language)é a linguagem de marcação usada para criar páginas da web. Ele é usado para estruturar o conteúdo de uma página, definindo elementos e tags que indicam como o conteúdo deve ser exibido. De modo mais simples, podemos dizer que o HTML consiste no esqueleto de uma página web, utilizada juntamente com CSS, para aplicarmos estilo à nossa página, e JavaScript para a lógica e funcionamento do site.

História do HTML

A história do HTML remonta aos primórdios da web. Foi criado por Tim Berners-Lee na década de 1990 como parte do projeto da World Wide Web. Desde então, passou por várias versões e evoluções.

Comunicação Client X Server

A comunicação entre cliente e servidor é fundamental para o funcionamento da web. O HTML é a linguagem usada pelo cliente para solicitar e exibir conteúdo web, enquanto o servidor fornece esse conteúdo em resposta às solicitações do cliente.

Na imagem acima podemos observar de modo mais claro, como ocorre a comunicação client-server. Os Clients são compostos pelos diversos dispositivos existentes que possuem acesso a internet (desktops, smartphones, notebooks, tablets, dentre outros), já os server são máquinas de grande porte que tem como função hospedar um site ou aplicação.

Quando um usuário (client) acessa um site, o navegador faz uma solicitação ao servidor(servers), que por sua vez, envia uma resposta ao navegador, que é exibida ao usuário.
